\doxysubsection{Detailed Description}
Interface for file handling services, defining methods for reading, saving, and exporting {\ttfamily .maso} files across different platforms.

The {\ttfamily File\+Service} class provides functionalities for managing {\ttfamily .maso} files. This includes reading a {\ttfamily .maso} file, saving a {\ttfamily \doxylink{class_maso_file}{Maso\+File}} object to the file system, and interacting with the user for file selection.

The {\ttfamily File\+Service} class provides functionalities for managing {\ttfamily .maso} files. This includes reading, decoding, saving, and picking {\ttfamily .maso} files across platforms.
