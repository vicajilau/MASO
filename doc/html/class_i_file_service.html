Detailed Description

Interface for file handling services, defining methods for reading, saving, and exporting .maso files across different platforms.

The FileService class provides functionalities for managing .maso files. This includes reading a .maso file, saving a MasoFile object to the file system, and interacting with the user for file selection.

The FileService class provides functionalities for managing .maso files. This includes reading, decoding, saving, and picking .maso files across platforms.
