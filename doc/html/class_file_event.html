Detailed Description

Abstract class representing the base event for file operations.

Event triggered when a file save is requested, with the file path and data.

Event triggered when a file is requested to be picked. This event can be used to initiate the file selection process.

Event triggered to reset the file state. This event can be used to clear any file-related data or state.
